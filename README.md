#Pink_TO_DO_LIST

#This is a simple to do list I made for anyone to use. The theme is pink becase I love pink and just thought considering it is a mainstream color now, then it would be used by a lot of people. 

#Features:
- you can add tasks
- you can delete tasks
- add your own categories
- quick suggestion of categories
- see completed tasks
- see uncompleted tasks
-sort tasks by name, category, completion status, oldest, and newest

#Installation:
-   Easy way: (Web Version)
    1. Copy the html code 
    2. Create a new file (notepad)
    3. Save as html
    4. Open it
- Python way: 
    1. Install python (if you dont have it)
    2. Create a python file (notepad) .py
    3. Copy ython file to the file
    4. Run it on command prompt or VS code
